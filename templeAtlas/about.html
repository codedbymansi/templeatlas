About the Project

BCA Project | Digital Heritage Analytics

Our Mission & Vision

TempleAtlas is a digital initiative to preserve India’s architectural legacy. We aim to bridge the gap between traditional history and modern data science by creating a navigable, interactive heritage ecosystem.

Key Project Objectives

✓
Geospatial Mapping: Successfully plot 50+ heritage sites with precise coordinates using Leaflet.js.
✓
Data Analytics: Implement real-time calculation of dynasty and state-wise temple distributions.
✓
User Personalization: Develop a LocalStorage-based CRUD system for "Favorite" collections.
✓
Responsive UI: Ensure 100% compatibility across mobile, tablet, and desktop viewports.
